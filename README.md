# Vibe Coder Learn - 2026 Vibe Coder 进阶之路

> **从零到编排者：为零代码基础的人设计的 AI 时代软件构建课程**

你不需要学会写代码，你需要学会**指挥 AI 写代码**。

---

## 这是什么

一套完整的学习资源，帮助非程序员通过 AI 工具（Claude Code、Cursor、Replit Agent 等）构建真实的软件产品。

核心理念：**品味比技能更重要**。你可以不会写代码，但你需要能判断代码方案的好坏。

## 从哪里开始

```
如果你是完全的新手：
  → 从 LEARNING-PATH.md 开始（12周结构化学习路径）

如果你想直接看课程内容：
  → CURRICULUM.md（主线7个递进式项目）

如果你想直接练品味：
  → TASTE-CHALLENGES.md（153个品味挑战）
```

## 课程结构

### 四层能力模型

```
第四层：编排层  — 多代理协作、规格驱动开发（SDD）
第三层：治理层  — 安全审计、代码审查、成本管理
第二层：通信层  — CLAUDE.md、SPEC.md、提示工程
第一层：基础层  — 互联网原理、数据库、部署
```

### 7个主线项目

| # | 项目 | 你将构建的东西 |
|---|------|-------------|
| P1 | 数字名片 | 个人品牌网站 |
| P2 | 灵感捕手 | 智能书签管理器 |
| P3 | 问卷星球 | 互动表单系统 |
| P4 | 圈子 | 带认证的社区平台 |
| P5 | 微店 | 支付集成的电商产品 |
| P6 | AI 伙伴 | AI 驱动的应用 |
| P7 | 全栈工厂 | 生产级系统 |

### 12个品味维度（153个挑战）

架构直觉 (15) · UX/产品品味 (18) · 安全嗅觉 (12) · 提示工程 (15) · 成本与商业 (15) · 代码审查 (10) · 营销与增长 (15) · 数据与分析 (10) · AI/LLM 集成 (15) · 沟通与文档 (10) · 运维与可靠性 (10) · 性能优化 (8)

## 记住这些数字

- **45%** 的 AI 代码引入安全漏洞（Veracode 2025）——永远验证
- **80%** 是 AI 能轻松完成的——剩下的20%是你的价值
- **19%** 更慢——如果不理解代码就盲目使用 AI
